## 何時讀這份

當需求是前端 UI 設計、複雜 artifact 組裝、或本地 Web UI 自動化驗證時，先讀這份分類檔。

## Skill 細節入口

- [frontend-design](../skills/frontend-design/SKILL.md) — 生產級 UI 與強設計感前端
- [web-artifacts-builder](../skills/web-artifacts-builder/SKILL.md) — 多組件 artifact、狀態管理與 shadcn/ui
- [webapp-testing](../skills/webapp-testing/SKILL.md) — Playwright 操作、截圖、console / network 檢查

[返回開頭](#快速導覽)

## 注意事項

- 若需求核心是「設計」，先看 `frontend-design`；若核心是「複雜 artifact 工程」，先看 `web-artifacts-builder`。
- 若是驗證已存在的本地 Web app，而不是建 UI，本分類通常直接走 `webapp-testing`。
